addBungeeCommand

open fun addBungeeCommand(bungeeCommand: BungeeCommands, playername: String, argument: String): Boolean
Content copied to clipboard

Adds "bungee" command. It's not a standard command, executed by the server,but sent as net.minecraft.network.protocol.game.ClientboundCustomPayloadPacket to player and caught by proxy.

Return

true if command was added successfully (if enabled in config), otherwise false
